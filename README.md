## Abstract

This presentation examines how Assam's historical trajectory—from the pre-colonial Paik system through British colonization—created a path dependency that inhibited entrepreneurial development and fostered a state-dependent salaried class. The analysis draws on theoretical frameworks from Bishnupriya Gupta (service-led growth) and Pranab Bardhan (dominant proprietary classes) to explain persistent economic stagnation.

## Key Topics

- Pre-Colonial Ahom Paik System
- Colonial Rupture and Monetization
- Educational Bias and Rent-Seeking
- Ethno-Nationalism as Rent Protection
- Fiscal Crisis and Development Constraints
